# E-Commerce SQL System

This project demonstrates a simple e-commerce database management system using MySQL.

## Features
- Database schema for Customers, Products, and Orders.
- Database Normalization (creation of an `order_items` table).
- Complex queries including Joins, Aggregations (SUM, AVG), and Data Updates.

## How to Run
1. Open MySQL Workbench.
2. Copy the contents of `ecommerce_db_project.sql`.
3. Execute the script to build the database and see the results.

## Key Queries Included
- Top 3 highest-priced products.
- Total amount spent by each customer.
- Average order value.
- Customer retrieval based on order date.
